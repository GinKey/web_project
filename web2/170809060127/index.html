Main labrories

The main libraries for working with big data in python are pandas numpy and matplotlib. They are very easy to learn and contain a lot of useful features.

 

Pandas

Pandas is a fast, powerful, flexible and easy to use open source data analysis and manipulation tool, built on top of the Python programming language.

 

Numpy

NumPy is an open source library for the Python programming language. Features: support for multidimensional arrays; support for high-level mathematical functions designed to work with multidimensional arrays.

 

Matplotlib

Matplotlib is a comprehensive library for creating static, animated, and interactive visualizations in Python. Matplotlib makes easy things easy and hard things possible.
